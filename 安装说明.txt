CalcAny 数据分析工具 - 安装和使用说明
===========================================

📁 文件说明：
├── dist/CalcAny.exe          # 主程序文件（约50-80MB）
├── config.json               # 配置文件
├── sample.json               # 示例数据文件
├── 运行CalcAny.bat           # 快速启动脚本
├── README_EXE.md             # 详细使用说明
└── 安装说明.txt              # 本文件

🚀 快速开始：
1. 双击「运行CalcAny.bat」启动程序
   或者
   直接双击「dist/CalcAny.exe」

2. 如果被杀毒软件拦截：
   - 选择「允许运行」或「信任此程序」
   - 或将整个文件夹添加到杀毒软件白名单

🛡️ 防误杀建议：
• Windows Defender：设置 → 病毒和威胁防护 → 排除项
• 360安全卫士：木马查杀 → 信任区
• 腾讯电脑管家：病毒查杀 → 信任区
• 火绒安全：右键程序 → 加入信任区

📋 系统要求：
• Windows 7/8/10/11 (64位)
• 至少 100MB 可用磁盘空间
• 建议 4GB 以上内存

✨ 程序特性：
• 无需安装Python环境
• 独立运行，包含所有依赖
• 数据本地处理，不联网
• 可视化配置管理界面（菜单：工具 → 配置管理）
• 支持配置文件导入、导出和重置
• 完整的日志记录功能
• 内存优化，性能良好

🔧 配置管理功能：
1. 打开程序后，点击菜单栏「工具」→「配置管理」
2. 在配置管理窗口中可以：
   - 编辑当前配置项
   - 导入外部配置文件（JSON格式）
   - 导出当前配置到文件
   - 重置为默认配置
   - 保存配置更改
3. 配置更改后会自动刷新程序设置

❓ 常见问题：
1. 程序启动慢？
   → 首次运行需要解压，属于正常现象

2. 找不到配置文件？
   → 程序会自动创建默认配置

3. 数据处理慢？
   → 程序已内置优化，大文件建议分批处理

📞 技术支持：
如有问题请查看详细的 README_EXE.md 文件
或检查 logs/ 目录下的日志文件
